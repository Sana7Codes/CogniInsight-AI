# 🧠 CogniInsight AI

> **Plateforme d'analyse cognitive alimentée par l'IA** — un tableau de bord Streamlit qui
> classe les profils cognitifs avec KMeans, visualise les clusters en PCA 2D et génère des
> rapports personnalisés en français via l'API Claude (claude-opus-4-6).

---

## Français

### Description du projet

CogniInsight AI est une application d'analyse de données cognitives construite au-dessus de
[CogniReaction](https://github.com), un outil de tests cognitifs mesurant le temps de réaction,
la précision et les taux d'erreur. Ce projet ajoute une couche IA complète :

1. **Clustering KMeans** — 3 profils cognitifs détectés automatiquement : *Focused*, *Fatigué*, *Impulsif*
2. **Visualisation PCA 2D** — scatter plot interactif (Plotly) des clusters
3. **Tendances utilisateur** — graphiques d'évolution par session
4. **Rapport IA personnalisé** — 3 paragraphes en français générés par Claude (analyse, causes, recommandations)
5. **Export PDF** — rapport téléchargeable avec fpdf2

### Architecture

```
cogniinsight/
│
├── app.py               ← Dashboard Streamlit principal
│   ├── Chargement des données (CSV ou exemple intégré)
│   ├── Tab 1 : Clustering PCA scatter plot
│   ├── Tab 2 : Tendances par utilisateur (line charts + radar)
│   └── Tab 3 : Rapport IA + export PDF
│
├── clustering.py        ← Pipeline scikit-learn
│   ├── StandardScaler → KMeans(k=3) → label mapping
│   └── PCA(n=2) pour visualisation
│
├── report_generator.py  ← Appel API Anthropic
│   ├── Prompt structuré (métriques + profil + instructions FR)
│   └── claude-opus-4-6 avec adaptive thinking
│
├── pdf_export.py        ← Génération PDF (fpdf2)
│   ├── Header brandé, table de métriques, badge de profil
│   └── Corps du rapport avec mise en forme
│
├── sample_data.py       ← Données synthétiques réalistes
│   ├── 20 utilisateurs × 2-4 sessions ≈ 50 lignes
│   └── 3 clusters bien séparés pour KMeans
│
├── requirements.txt
├── Dockerfile
├── .env.example
└── README.md

Flux de données :
  CSV  ──►  get_user_aggregate()  ──►  run_clustering()  ──►  Plotly charts
                                             │
                                             ▼
                                   generate_report()  ──►  Claude API
                                             │
                                             ▼
                                   create_pdf_report()  ──►  PDF bytes
```

### Lancer en local

**Prérequis** : Python 3.11+, clé API Anthropic

```bash
# 1. Cloner et accéder au dossier
cd cogniinsight

# 2. Environnement virtuel
python -m venv .venv
source .venv/bin/activate        # Windows : .venv\Scripts\activate

# 3. Dépendances
pip install -r requirements.txt

# 4. Clé API
cp .env.example .env
# Éditez .env et ajoutez votre ANTHROPIC_API_KEY

# 5. Lancer
streamlit run app.py
```

L'application est disponible sur **http://localhost:8501**.

### Déployer sur Hugging Face Spaces

1. Créez un nouveau Space (type : **Streamlit**)
2. Uploadez tous les fichiers du dossier `cogniinsight/`
3. Dans **Settings → Repository secrets**, ajoutez :
   - `ANTHROPIC_API_KEY` = votre clé Anthropic
4. Le Space démarre automatiquement — aucune configuration supplémentaire

## English

### Project Description

CogniInsight AI is a cognitive data analysis platform built on top of CogniReaction (a
cognitive test web app measuring reaction time, accuracy, and error rates). It adds an AI
analysis layer:

1. **KMeans Clustering** — 3 auto-detected cognitive profiles: *Focused*, *Fatigued*, *Impulsive*
2. **PCA 2D Visualisation** — interactive Plotly scatter plot of clusters
3. **User Session Trends** — per-user line charts and radar plots
4. **Personalised AI Report** — 3-paragraph French report generated by Claude (analysis, causes, recommendations)
5. **PDF Export** — downloadable report via fpdf2

### Architecture

```
┌─────────────────────────────────────────────────────────────────┐
│                        Streamlit UI (app.py)                    │
│  ┌──────────────┐  ┌──────────────────┐  ┌────────────────────┐ │
│  │  Tab 1       │  │  Tab 2           │  │  Tab 3             │ │
│  │  Clustering  │  │  User Trends     │  │  AI Report + PDF   │ │
│  │  PCA 2D plot │  │  Line + Radar    │  │  Claude API call   │ │
│  └──────┬───────┘  └────────┬─────────┘  └─────────┬──────────┘ │
└─────────┼──────────────────┼──────────────────────┼────────────┘
          │                  │                        │
          ▼                  ▼                        ▼

                                               (fpdf2 PDF)
```

### Run Locally

**Prerequisites**: Python 3.11+, Anthropic API key

```bash
cd cogniinsight
python -m venv .venv
source .venv/bin/activate    # Windows: .venv\Scripts\activate
pip install -r requirements.txt

cp .env.example .env
# Edit .env: set ANTHROPIC_API_KEY=sk-ant-...

streamlit run app.py
# Open http://localhost:8501
```

### Docker

```bash
docker build -t cogniinsight .

```

### CSV Format

| Column | Type | Description |
|---|---|---|
| `user_id` | string | Unique user identifier |
| `session_number` | int | Session index per user (optional) |
| `session_date` | date | ISO date (optional) |
| `reaction_time_ms` | float | Mean reaction time in milliseconds |
| `accuracy_pct` | float | Accuracy percentage (0–100) |
| `error_rate` | float | Error rate percentage (0–100) |
| `n_trials` | int | Number of trials in session |
| `test_type` | string | Test name (optional) |
